# Nepal Tourism Dashboard Dataset — README

## Overview

This dataset contains tourist arrival and revenue records for Nepal across all months of 2024. It is structured for Excel-based dashboard development and analysis practice, covering visitor flows by country of origin, province, and destination. The workbooks are designed to support hands-on exercises in functions, pivot tables, conditional formatting, and KPI dashboard building.

---

## Repository Contents

| File | Type | Description |
|------|------|-------------|
| `nepal_tourism_dashboard_dataset.xlsx` | Primary Dataset | 500 tourist records with a project task list for guided analysis practice |
| `nepal_tourism_dashboard_turnover.xlsx` | Sorted / Analysis Workbook | Same records sorted by revenue (descending), with pre-built pivot summaries and turnover analysis by destination |

---

## Dataset Scope

- **Time period:** Full calendar year 2024 (January – December)
- **Granularity:** Individual transaction records (one row per tourist group arrival)
- **Records:** 500 rows
- **Countries covered:** Australia, China, France, Germany, India, Japan, UK, USA
- **Provinces covered:** Bagmati, Gandaki, Karnali, Koshi, Lumbini, Madhesh, Sudurpashchim
- **Destinations covered:** Chitwan, Everest Region, Kathmandu, Lumbini, Mustang, Pokhara

---

## Column Reference

| Column | Type | Description |
|--------|------|-------------|
| `Record_ID` | Integer | Unique identifier for each arrival record |
| `Date` | Date (YYYY-MM-DD) | Date of tourist arrival |
| `Year` | Integer | Calendar year of arrival (all records: 2024) |
| `Month` | String | Month name of arrival |
| `Country` | String | Tourist's country of origin |
| `Province` | String | Nepal province where the visit was recorded |
| `Destination` | String | Specific tourist destination visited |
| `Tourists` | Integer | Number of tourists in the group or batch (range: 53 – 1,200) |
| `Avg_Stay_Days` | Float | Average length of stay in days (range: 3.0 – 14.0) |
| `Revenue_USD` | Integer | Total revenue generated by the visit in US Dollars (range: $5,665 – $285,425) |

## Notes

- All revenue values are denominated in **US Dollars (USD)**.
- `Tourists` represents the count of visitors per record entry, not individual headcount; group sizes vary widely (53 to 1,200).
- The `Province` and `Destination` fields are independent — a destination such as Pokhara may appear under multiple provinces depending on the recorded itinerary entry.
- The turnover workbook is sorted by revenue (highest to lowest) to facilitate top-performer analysis and is otherwise identical in raw data to the primary dataset workbook.
- Records span all 12 calendar months of 2024, allowing seasonal analysis across the full year.
